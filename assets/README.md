# assets

This folder is for optional media.

## Ambient audio (optional)

Drop a file named **`ambient.mp3`** here to give the experience a soft
background soundtrack. The page already contains an `<audio>` element and a
`♪` toggle in the top-right corner — it will simply stay silent until the
file exists, so nothing breaks if you leave this empty.

Recommended: a quiet, looping ambient/drone track (low volume, no sudden
changes) to match the calm, cinematic mood.

```
assets/
└── ambient.mp3   ← add this yourself (not included)
```

Nothing else is required. The whole site works fully offline without any
files in this folder.
